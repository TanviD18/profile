# profile

Static profile site published with GitHub Pages.

## Publish from GitHub

This repository now includes a GitHub Actions workflow at `.github/workflows/deploy.yml` that deploys the site to GitHub Pages whenever you push to `main`.

To enable publishing in GitHub:

1. Open the repository on GitHub.
2. Go to Settings > Pages.
3. Under Source, select GitHub Actions.
4. Push your latest changes to `main`, or run the `Deploy static site to Pages` workflow manually from the Actions tab.

## Site URL

- With the current repository name, the site will publish at `https://tanvid18.github.io/profile/`.
- If you want this to be your main profile site at `https://tanvid18.github.io/`, the repository name must be `TanviD18.github.io`.
